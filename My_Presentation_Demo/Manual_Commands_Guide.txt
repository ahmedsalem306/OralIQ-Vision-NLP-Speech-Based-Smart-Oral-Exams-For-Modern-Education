===============================================================
        أوامر التشغيل اليدوية للمشروع في يوم المناقشة
===============================================================
إذا أردت تشغيل المشروع بنفسك بكتابة الأوامر يدوياً في الـ Terminal بدلاً من الضغط على الأزرار الجاهزة، فاتبع هذه الخطوات بالترتيب:

أولاً: لتشغيل موقع الفرونت إند (Frontend)
------------------------------------------
1. افتح الـ Terminal أو My_Presentation_Demo في VS Code
2. ادخل إلى مجلد الفرونت إند بكتابة:
   cd frontend

3. في أول مرة فقط (لتحميل المكتبات الناقصة، الخطوط الحمراء):
   npm install

4. لتشغيل السيرفر والموقع:
   npm run dev

5. سيظهر لك رابط مثل (http://localhost:5173)، لا تضغط عليه مباشرة (لأنه سيحولك لصفحة الدخول التي لن تعمل بدون باك إند)، بل استخدم هذه الروابط المباشرة لعرض الصفحات للدكتورة:
    - رابط شاشة الطالب (شاشة تتبع الوجه): http://localhost:5173/exam/start
    - رابط لوحة درجات الطالب (الدرجات الشخصية): http://localhost:5173/dashboard/grades
    - لوحة تحكم الدكتور (النتائج): http://localhost:5173/dashboard/results
    - صفحة الحضور والنزاهة: http://localhost:5173/dashboard/attendance
    - صفحة التحليلات البيانية: http://localhost:5173/dashboard/analytics
    - لمشاهدة واجهة إضافة الأسئلة: http://localhost:5173/dashboard/questions
   
   (ملاحظة هامة جداً: الروابط دي عشان تعرض شكل الشاشات "UI" وشغل الكاميرا "AI" قدام الدكتورة، لكن البيانات اللي فيها هتكون فاضية أو مش بتسجل لأن مفيش Back-end أو Database شغال يكلمها).


ثانياً: لتشغيل موديل المراقبة والموبايل (Face Tracking Model)
------------------------------------------------------------
1. افتح Terminal جديد.
2. ادخل إلى المجلد الأساسي:
   cd d:\interview-ai\My_Presentation_Demo

3. ستحتاج إلى تفعيل بيئة البايثون الوهمية (Virtual Environment) إذا كانت لديك في المشروع القديم:
   ..\venv\Scripts\activate.bat
   
   (ملاحظة: إذا نقلت المجلد على جهاز الدكتورة ولا يوجد لديك Python مسطب، سيصعب تشغيل هذا الموديل ويُفضل الاكتفاء بالفرونت إند الذي يعمل في المتصفح).

4. أخيراً، تشغيل كود المراقبة:
    .\run_python_model.bat

5. الكاميرا ستُفتح فوراً. لإغلاقها، اضغط على رز "q" في الكيبورد أثناء التحديد على شاشة الكاميرا.
